# 小安助手 - Android App 完整部署指南

## 项目概述

**小安助手** 是一款面向安卓平板的 AI 语音助手 App，具备以下功能：
- ✅ 登录功能（任意账号密码均可登录）
- ✅ 语音唤醒（说「小安小安」自动唤醒）
- ✅ TTS 文本转语音（唤醒后说「你好，我在」）
- ✅ ASR 语音识别（识别内容实时显示在屏幕）
- ✅ 持续语音对话

## 方法一：GitHub Actions 在线自动编译（推荐，无需安装任何工具）

### 第一步：上传代码到 GitHub

1. 前往 [https://github.com](https://github.com) 注册/登录
2. 点击右上角 **+** → **New repository**
3. 仓库名填：`XiaoAnApp`，选择 **Public**（免费用 Actions）
4. 点击 **Create repository**

### 第二步：上传项目文件

**方式 A（推荐）：用 GitHub Desktop**
1. 下载安装 [GitHub Desktop](https://desktop.github.com/)
2. File → Add Local Repository → 选择 `C:\Users\韦叠凤\WorkBuddy\Claw\XiaoAnApp`
3. Commit → Push

**方式 B：直接网页上传**
1. 进入你的仓库页面
2. 将 `XiaoAnApp` 文件夹里的所有文件拖拽上传
3. 注意保持文件夹结构

### 第三步：等待自动编译

1. 上传完成后，点击仓库的 **Actions** 标签页
2. 可以看到 "Build XiaoAn APK" 工作流正在运行（约 3-5 分钟）
3. 运行完成后，点击该次运行记录
4. 在底部 **Artifacts** 区域下载 `XiaoAn-debug-apk`

### 第四步：安装到平板

1. 解压下载的 zip，得到 `app-debug.apk`
2. 通过 USB 数据线或微信传输到平板
3. 在平板上打开文件管理器，找到 APK 文件
4. 点击安装（需要在设置中允许"安装未知来源应用"）
5. 安装完成，打开「小安助手」

## App 使用说明

### 登录
- 任意输入用户名和密码，点击登录即可进入

### 语音唤醒
1. 进入主界面后，App 自动开始监听
2. 说出「**小安小安**」→ 平板会用语音回答「你好，我在」
3. 唤醒后进入对话模式，可以直接继续说话

### 语音对话
- 唤醒后，你说的所有内容会被 ASR 识别并显示在屏幕上
- 系统会给出简单的语音回复
- 说「**再见**」或「**拜拜**」可以退回唤醒等待状态

### 按钮说明
| 按钮 | 功能 |
|------|------|
| 开始说话 / 停止监听 | 手动控制麦克风监听状态 |
| 清空记录 | 清空对话历史 |

## 常见问题

**Q: 语音识别不工作？**
A: 需要在设置中允许麦克风权限。部分安卓设备需要安装 Google 语音服务（国内平板可能需要手动安装）

**Q: TTS 没有声音？**
A: 检查平板音量，并确保系统已安装中文 TTS 语音包（设置 → 无障碍 → 文字转语音）

**Q: 说「小安小安」没有反应？**
A: 说清楚一点，注意普通话发音。可以尝试多说几次。国内设备语音识别引擎默认需要网络。
